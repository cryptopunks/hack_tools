# Installation
## With git
```sh
$ git clone https://github.com/shawnduong/PXEnum
```
## With wget
```sh
$ wget https://raw.githubusercontent.com/shawnduong/PXEnum/master/PXEnum.sh
```

# Usage
```sh
$ sh PXEnum.sh
```
